1. 이용권 동시 사용 불가

Item Plan을 포함해 모든 이용권은 기본적으로 동시사용이 어렵습니다. 먼저 구입한 이용권의 기간이 남은 상태에서 추가로
이용권을 구입할 경우 먼저 구매한 이용권은 대기 상태로 전환됩니다.

ex) A 이용권 구매 후 B 이용권을 구매한 경우, B 이용권의 기간 종료 후 A 이용권의 남은 기간 사용가능
(단, Plan 업그레이드 및 E-GPT 추가구매는 아래 내용을 참고해주시기 바랍니다.)

2. Basic Plan 이용자가 Premium Plan으로 업그레이드 하고자 할 때

Basic Plan에 한해 Premium Plan으로 업그레이드 가능합니다. Basic Plan 이용자가 이용권 구매 후 14일 이전의 경우 차액(20,000원)만큼 Premium Plan으로 업그레이드 가능.(단 변경 후 환불 불가)

ex) 7일 Basic Plan 사용 후 업그레이드 시 23일간 Premium Plan 사용 가능
Basic Plan 이용자가 이용권 구매 후 15일 이후의 경우 Premium Plan 이용권 구매가 가능하며 Premium Plan 이용기간 종료 후 남은 기간만큼 Basic Plan 이용권을 이용할 수 있습니다.(단 구매 후 환불 불가)

ex) 15일 Basic Plan 사용 후 구매 시 1개월 Premium Plan 사용 후 15일 Basic Plan 사용 가능

3. E-GPT 추가구매 관련

Premium Plan 이용자에 한해 20,000원 추가 결제 시 이용기간 내 200건을 추가 사용할 수 있습니다. (Basic Plan 이용권 구매자는 E-GPT 구매 불가)
