SoonBeom Kwon

Multimodal interaction
의료 이미징
Robotics

이화여자대학교 공과대학 컴퓨터 인공지능 연구실은 인공지능을 이용한 의료 영상 분석에 중점을 두고 있습니다. 저희는 딥러닝과 머신 러닝 기법을 활용하여 MRI 이미지에서 정밀한 정보를 추출하고 진단하는 방법을 개발하고 있습니다.

이런 연구는 실제 의료 현장에서 더 빠르고 정확한 진단을 가능하게 하며, 특히 눈과 같은 복잡한 기관의 질병 감지에 큰 도움을 줍니다. 이는 의료 기기 제조 업체들이 보다 고도화된 진단 기기를 개발할 수 있는 기초를 마련해주며, 의료 데이터 분석 방법의 진보를 촉진시키는 중요한 기여를 하고 있습니다.

약하게 감독된 객체 위치 지정/감지

Weakly supervised object localization / detection

WSOL/D의 정확도와 효율성을 높여서 더 정밀한 객체 탐지를 위한 알고리즘 개발과 복잡한 배경을 가진 이미지에서도 객체를 효과적으로 분리할 수 있는 기술에 초점을 맞추고 있습니다. 이를 통해, 실세계 데이터에 대한 모델의 적용성을 향상시키고, 다양한 실용적 분야에서의 활용 가능성을 넓히는 것을 목표로 하고 있습니다.

강력한 물체 감지를 위한 데이터 증강/테스트 시간 조정

Data augmentation / Test time adaptation for robust object detection

데이터 증강은 이미지 변형을 통해 모델의 일반화 능력을 높이고, 테스트 시간 적응은 모델이 실제 환경 데이터에 즉시 적응하도록 합니다. 이 기술들은 변화하는 조건에서도 정확도를 유지하는 능력이 기존 방식보다 우수합니다. 이 기술이 더 적은 데이터와 정보로도 변화에 빠르게 적응하며, 더 높은 정확도를 달성할 수 있도록 발전시키는 것이 목표입니다.

오디오 유도에 의한 약하게 감독된 물체 감지

Audio-guided weakly supervised
object detection

오디오와 비디오 데이터를 결합하여 객체 탐지의 정확도를 향상시키는 다양한 방법을 연구하고 있습니다. 이를 통해 멀티모달 데이터를 활용하는 새로운 접근법을 개발하고, 오디오 데이터가 객체 탐지의 성능을 어떻게 개선할 수 있는지에 대한 이해를 높이고자 합니다.
